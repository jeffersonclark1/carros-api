# Simple CRUD API

Simple CRUD API server that connects to a database (mongoDB) and has operations for:

- Create
- Read
- Update
- Delete

### Instalation

Clone this repo and install the dependencies

```sh
$ git clone https://github.com/lamecksilva/Simple_CRUD_API.git
$ cd Simple_CRUD_API/
$ npm i
$ docker-compose up
```
